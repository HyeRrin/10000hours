# 1만시간의 법칙
## 1. 소개
### 🎯 목표
- 1만시간의 법칙은 어떤 분야의 전문가가 되려면 최소한 1만 시간 정도의 훈련이 필요하다는 법칙이다.
- 목표를 달성하기 위해 자신이 투자할 수 있는 시간을 입력하면, 며칠을 노력해야 1만시간을 달성해 목표에 가까워질 수 있는지를 계산해준다.

### 🔨 기능
- 반응형 웹 (Responsive Web)
- 모달창 (Modal Window)
- 폼 (Form)
